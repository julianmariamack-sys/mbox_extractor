📬 MBOX Extractor

A lightweight offline tool for extracting emails and attachments from .mbox files, with optional HTML export and GUI support.

✨ Features

  • 📎 Extract all attachments from MBOX files
  • 📄 Export emails as HTML (single or full archive)
  • 📁 Organize by Year → Month structure
  • 👤 Optional grouping by sender
  • 🖥️ Simple GUI included

⚙️ Installation

git clone https://github.com/yourname/mbox-extractor.git
cd mbox-extractor
python gui.py

No external dependencies required (pure Python).

💻 CLI Usage

Basic extraction

python extract.py mailbox.mbox

Export ALL emails as HTML

python extract.py mailbox.mbox --html-all

Export only emails with attachments

python extract.py mailbox.mbox --html

Filter attachments

python extract.py mailbox.mbox --types pdf jpg png

Organize by sender

python extract.py mailbox.mbox --by-sender

Combine options

python extract.py mailbox.mbox --html-all --by-sender --types pdf jpg

🖥️ GUI Usage

python gui.py

Features:

  • File picker for MBOX
  • Output folder selector
  • HTML export toggles
  • Sender grouping option
  • Attachment filtering

📁 Output Structure

output/
  2024/
    January/
      email_1.html
      invoice.pdf

With sender grouping:

output/
  2024/
    January/
      Gmail/
        email.html

📌 Use Cases

  • Email backups
  • Invoice extraction
  • Legal archiving
  • Data analysis preparation

⚠️ Limitations

  • No inline image (CID) embedding in HTML
  • No email threading reconstruction
  • Large mailboxes may take time to process
